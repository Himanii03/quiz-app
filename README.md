🧠 Quiz App (Python)

A simple command-line quiz application built using Python that reads questions from a JSON file, takes user input, and calculates the final score.

🚀 Features
Loads questions from a JSON file
Multiple-choice question format
Score tracking system
Displays final result at the end

🛠️ Tech Stack
Python 3.x
JSON (for storing questions)

▶️ How to Run
python quiz.py
📁 Project Structure
quiz-app/
│
├── quiz.py         # Main application file
├── data.json       # Questions database
└── README.md

📌 Example Question Format (data.json)
[
  {
    "question": "What is the capital of France?",
    "options": ["Paris", "Berlin", "Rome", "Madrid"],
    "answer": "Paris"
  }
]

📊 Output
User selects an option (1–4)
Score increases for correct answers
Final score displayed like: 3/5
